CONFESSHUB - HTML/CSS/JAVASCRIPT VERSION
=========================================

This is a standalone HTML/CSS/JavaScript version of the ConfessHub application.

FILES INCLUDED:
- index.html (Home page with hero and latest confessions)
- confessions.html (All confessions with filters)
- share.html (Share confession form)
- view-confession.html (Individual confession view with comments)
- about.html (About page)
- rules.html (Community rules page)
- styles.css (Custom styling)
- app.js (JavaScript functionality)

HOW TO USE:
1. Simply open index.html in any modern web browser
2. All pages are fully functional with Bootstrap 5
3. No server or build process required
4. Click around to navigate between pages

FEATURES:
- Responsive design (mobile-friendly)
- Bootstrap 5 components
- Font Awesome icons
- Interactive comment system
- Like/unlike functionality
- Form validation
- Clean purple theme

BROWSER COMPATIBILITY:
Works in all modern browsers (Chrome, Firefox, Safari, Edge)

Note: This is a demo application. Confessions are stored locally in JavaScript
and will reset on page reload. To make it fully functional, you would need to
connect it to a backend server/database.
